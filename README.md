# Fedora Hyprland Setup

Hyprland dotfiles for Fedora with Waybar, Rofi, Kitty, Matugen-driven theming, and a `wallgen` helper (generated during install) to rotate wallpapers and recolor the desktop.

## What you get
- Hyprland configuration with autostart for Waybar and `swww` in [.config/hypr/hyprland.conf](.config/hypr/hyprland.conf)
- Matugen templates that recolor Hyprland, Kitty, Rofi, and Waybar in [.config/matugen/templates](.config/matugen/templates/hyprland-colours.conf)
- Waybar layout and styling tuned for Material-derived colors in [.config/waybar](.config/waybar/config.jsonc)
- Rofi launcher theme with wallpaper backdrop in [.config/rofi/launchers/type-7/style-7.rasi](.config/rofi/launchers/type-7/style-7.rasi)
- `wallgen` helper to set a wallpaper, run Matugen, and sync Rofi’s background (the installer writes it to `~/.local/bin/wallgen`)

## Quick start
1) From a fresh Fedora 39+ install, clone this repo and run the installer as your regular user (do **not** run as root):
   ```bash
   chmod +x install.sh
   ./install.sh
   ```

2) Pick the **Hyprland** session in your display manager after log out.

3) Set a wallpaper and recolor the system:
   ```bash
   wallgen ~/Pictures/Wallpapers/your-image.jpg
   ```

The installer prints a log path (e.g., `/tmp/fedora-hyprland-XXXX.log`) you can share if something fails.

## Installer switches
Pass these as environment variables before running `./install.sh`.
- `INSTALL_CHROME=true|false` (default true)
- `INSTALL_VSCODE=true|false` (default true)
- `INSTALL_MATUGEN_METHOD=cargo|skip` (default cargo)
- `ENABLE_BATTERY_TUNING=true|false` (default true) — controls TLP, Wi-Fi powersave, and Hypr refresh automation.
- `MONITOR_NAME`, `MONITOR_RESOLUTION`, `MONITOR_REFRESH_BAT`, `MONITOR_REFRESH_AC` — monitor ID and refresh rates for the Hypr auto-switcher.
- `GPU_VENDOR=auto|nvidia|amd|intel|unknown` — `auto` detects via `lspci`; `nvidia` enables RPM Fusion and installs akmod drivers. Set `ENABLE_NVIDIA_ENV=false` to skip Wayland environment overrides.
- `DRY_RUN=true` — rsync-based config copies/backups run in `--dry-run` mode with checksums so you can see planned changes.

## Notes and adjustments
- Waybar network interface defaults to `wlp4s0`; update `"interface"` in [.config/waybar/config.jsonc](.config/waybar/config.jsonc#L24) to match your device.
- Default apps in Hyprland bindings: Kitty, Nautilus, Google Chrome, and VS Code. Change `$browser`, `$fileManager`, and `$ide` in [.config/hypr/hyprland.conf](.config/hypr/hyprland.conf#L26-L31) if you prefer alternatives.
- Wallpaper paths inside Matugen output and the Rofi theme expect files under `~/Pictures/Wallpapers`. The installer creates this folder for you.
- The installer backs up any existing configs it overwrites to a timestamped folder in your home directory.
- If your laptop uses NVIDIA and you set `GPU_VENDOR=nvidia`, the installer enables RPM Fusion and installs the NVIDIA akmod driver plus Wayland-friendly environment overrides.
- Battery tuning is on by default; set `ENABLE_BATTERY_TUNING=false` if you want to skip TLP/refresh-rate tweaks or if you’re on unsupported hardware.
